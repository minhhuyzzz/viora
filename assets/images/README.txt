Thư mục ảnh sản phẩm VIORA
============================
Đặt ảnh sản phẩm vào đây theo đúng tên trong products.js

Ví dụ từ data:
  /images/h1.jpg     → assets/images/h1.jpg
  /images/h1.1.jpg   → assets/images/h1.1.jpg
  ...

Định dạng hỗ trợ: .jpg, .jpeg, .webp, .png
Kích thước khuyến nghị: 800×1066px (tỷ lệ 3:4)
